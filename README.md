# Mô tả chương trình

# Yêu cầu môi trường

- `ruby 2.7.x`
- `Rails 6.1.x`

# Cài đặt chương trình

- `bundle install`
- `yarn install --check-files`
- `rake db:migrate`
- `bundle exec rake assets:precompile`

# Chạy chương trình

- `./bin/dev`
